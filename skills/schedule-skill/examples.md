# 日程管理技能使用示例

## 示例 1：创建日程

**用户输入：**
```
明天下午3点到4点有个团队会议，在会议室A
```

**智能体行为：**
1. 解析时间：明天 → 2026-03-14，下午3点 → 15:00
2. 调用 `add_event`：
   ```json
   {
     "title": "团队会议",
     "start_time": "2026-03-14 15:00",
     "end_time": "2026-03-14 16:00",
     "location": "会议室A"
   }
   ```

## 示例 2：查询日程

**用户输入：**
```
这周有什么安排？
```

**智能体行为：**
1. 计算本周日期范围
2. 调用 `query_event`：
   ```json
   {
     "start_date": "2026-03-09",
     "end_date": "2026-03-15"
   }
   ```

## 示例 3：修改日程

**用户输入：**
```
把明天的会议改到后天下午2点
```

**智能体行为：**
1. 先查询明天的会议获取 event_id
2. 调用 `modify_event`：
   ```json
   {
     "event_id": "abc12345",
     "start_time": "2026-03-15 14:00",
     "end_time": "2026-03-15 15:00"
   }
   ```

## 示例 4：解析文档

**用户输入：**
```
帮我解析这个日程安排：
- 3月15日上午10点项目评审
- 3月20日下午2点客户会议
```

**智能体行为：**
1. 调用 `parse_document`：
   ```json
   {
     "content": "- 3月15日上午10点项目评审\n- 3月20日下午2点客户会议",
     "source_type": "text"
   }
   ```
2. 根据解析结果创建日程
