# 日程管理技能

## 概述

此技能帮助智能体管理用户的个人日程安排，包括日程的创建、查询、修改和删除操作。

## 能力

- **创建日程**：从自然语言中提取时间、地点、描述等信息，创建日程
- **查询日程**：按日期范围、关键词查询日程
- **修改日程**：修改已有日程的时间、地点、描述等信息
- **删除日程**：删除指定的日程
- **解析文档**：从 Markdown 文件或聊天记录中提取日程信息

## 使用场景

1. 用户说"明天下午 3 点开会"时，自动创建日程
2. 用户问"这周有什么安排"时，查询本周日程
3. 用户说"把周一下午的会议改到周三"时，修改日程
4. 用户上传会议记录文档时，提取其中的日程安排

## 工具列表

| 工具名 | 功能 |
|--------|------|
| add_event | 添加日程 |
| modify_event | 修改日程 |
| query_event | 查询日程 |
| delete_event | 删除日程 |
| parse_document | 解析文档提取日程 |

## 注意事项

- 时间格式统一为 `YYYY-MM-DD HH:MM`
- 创建日程时检查时间冲突
- 相对时间（如"明天"、"下周三"）需要转换为具体日期
- 修改日程时需要先查询获取 event_id
- 删除操作前需要用户确认
